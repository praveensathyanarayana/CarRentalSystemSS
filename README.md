## Project Overview

This Car Rental System is a comprehensive demonstration of Object-Oriented Programming principles, SOLID design patterns, and modern software architecture practices implemented using Spring Boot and Java.

### Key Features
- ✅ All 4 OOP Pillars (Encapsulation, Abstraction, Inheritance, Polymorphism)
- ✅ All 5 SOLID Principles
- ✅ 5 Design Patterns (Repository, Strategy, Factory, DTO, CQRS)
- ✅ Domain-Driven Design concepts
- ✅ Clean Architecture / Layered Architecture
- ✅ 100% Testable with Dependency Injection
